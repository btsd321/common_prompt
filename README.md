# common_prompt

# 浏览内容，生成项目概述文档
1. copilot
    检查工程目录下.github文件夹中是否有概述文件没有就创建。
    浏览项目内容，总结概述文档，如果项目比较大，要先拆分理解各模块作用，创建各模块的概述文档，主文档引用模块文档。
    编写文档时要先列大纲，再逐段填充
    分步骤进行每进行一步整理一下已有知识，删除无用知识
2. claude
    检查工程目录下.claude文件夹中是否有概述文件没有就创建。
    浏览项目内容，总结本项目概述文档，如果项目比较大，要先拆分理解各模块作用，创建各模块的概述文档（按官方推荐的方式进行创建和管理），主文档引用模块文档。
    编写文档时要先列大纲，再逐段填充
    分步骤进行每进行一步整理一下已有知识，删除无用知识
